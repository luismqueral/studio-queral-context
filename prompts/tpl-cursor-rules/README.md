# Cursor Rules Templates

This directory contains template .cursorrules files for different project types and contexts.

## Available Templates

- **brand2.mdc** - Brand-focused rules for maintaining NYT visual consistency
- **combined.mdc** - Comprehensive ruleset combining multiple design system aspects
- **storylist.mdc** - Rules optimized for story listing and content presentation interfaces
- **swift-ui.mdc** - Rules adapted for Swift UI development with NYT design principles
- **tpl-tokens.mdc** - Token-focused rules emphasizing design system compliance

## Usage

Copy the appropriate .mdc file content to your project's .cursorrules file. Modify the rules as needed to match your specific project requirements while maintaining adherence to NYT design system standards.
